# codec-technologies-projects
# Codec Technologies Internship Projects

This repository contains two machine learning projects completed during the internship:

---

## 📘 Project 1: Handwritten Digit Recognizer
- **Dataset:** MNIST (built-in Keras dataset)
- **Model:** Convolutional Neural Network (CNN)
- **Accuracy:** ~98%
- **Notebook:** `HandwrittenDigitRecognizer_(MNIST).ipynb`
- **Results:** See `results/` folder for accuracy plots and sample predictions.

---

## 📘 Project 2: Spam Email Classifier
- **Dataset:** SMS Spam Collection Dataset (Kaggle)
- **Model:** Multinomial Naive Bayes
- **Accuracy:** ~95%
- **Notebook:** `SpamEmailClassifier.ipynb`
- **Results:** See `results/` folder for confusion matrix and accuracy screenshots.

---

## 📘 How to Run
1. Open notebooks in Google Colab.
2. For Project 2, upload `spam.csv` dataset into Colab.
3. Run all cells step by step.
4. Push notebook updates to GitHub via **File → Save a copy in GitHub**.
